# ORBITAL Spatial Intelligence

A cinematic WebGL landing experience for mission-scale spatial intelligence: Earth scale, orbit context, and decision-room storytelling.

Live site: https://twk0672005.github.io/orbital-spatial-intelligence/

## Contents

- `index.html` - standalone static WebGL site
- `assets/` - Earth day, night, and cloud textures
- `vendor/three.module.js` - local Three.js module for stable GitHub Pages loading

## Notes

This repository is a public deployment package. Development reports, backups, temporary QA files, and local verification artifacts are intentionally excluded.
